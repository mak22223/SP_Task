## Сборка проекта

Для сборки воспользуйтесь следующим набором команд, находясь в корне проекта:
```
mkdir build
cd build
cmake ..
make
```

## Запуск мастера

Находясь в директории build
```
./src/master/master
```

## Запуск воркера

Находясь в директории build
```
./src/worker/worker
```
